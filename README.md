🍳 Recipe Finder

Recipe Finder is a dynamic web application that helps users discover delicious recipes based on the ingredients they already have. Simply enter ingredients, and the app fetches relevant recipes using the Spoonacular API.

🚀 Features

🔍 Search recipes by ingredients

📦 Fetch real-time data from Spoonacular API

🖼 Display recipe images and titles

📖 View detailed instructions in modal popup

🍪 Saves last searched ingredients using cookies

🎯 Smooth auto-scroll to results

📱 Fully responsive design

🛠 Tech Stack

HTML5

CSS3

JavaScript (Vanilla JS)

Spoonacular API
